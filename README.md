# psn scraper

### Setting up
----

Cloning the repo:
```sh
git clone git@github.com:jamesesguerra/psn_scraper.git
```

Making a virtual env:
```sh
python -m venv venv
```

Activating the env:
```sh
# windows
\venv\Scripts\activate

#unix
source venv/bin/activate
```

Installing Scrapy:
```sh
pip install scrapy
```

Change dir:
```sh
cd psn_scraper
```

### usage
----

Cd into `article_scraper`:
```sh
cd article_scraper
```

Run `main.py`:
```sh
python main.py
```
